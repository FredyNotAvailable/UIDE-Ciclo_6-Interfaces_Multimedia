# Canvas Experimentation - Partículas Interactivas

## Descripción
Esta es una aplicación web interactiva desarrollada con HTML5 Canvas y Vanilla JavaScript (sin librerías externas). Muestra un sistema de partículas que se mueven de forma autónoma y reaccionan a la presencia del mouse del usuario.

## Interacción Elegida: Repulsión
Las partículas se mueven libremente por el lienzo. Cuando el usuario mueve el cursor sobre el canvas, las partículas cercanas "huyen" del cursor, creando un efecto de dispersión o campo de fuerza.

## Cómo probarlo
1. Abre el archivo `index.html` en cualquier navegador web moderno.
2. Mueve el **mouse** sobre el área oscura.
3. Observa cómo los puntos (partículas) se alejan de tu puntero.
4. Prueba cambiar el tamaño de la ventana del navegador; el sistema se adaptará y reiniciará la distribución de puntos.

## Estructura de Archivos
- `index.html`: Estructura base.
- `style.css`: Estilos visuales.
- `script.js`: Lógica del sistema de partículas y animación.
